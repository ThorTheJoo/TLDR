## 🚀 **Quick Start Commands**

### **Start OAuth Server**
```bash

start-oauth-server.bat

# PowerShell
.\start-oauth-server.ps1

# Manual
$env:PATH += ";C:\Program Files\nodejs"
node oauth-callback-server.js
```
